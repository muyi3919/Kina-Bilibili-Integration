# 🎬 Kina Bilibili Integration

一个为 WordPress 打造的 Bilibili 集成插件，支持展示个人资料、追番列表和实时直播状态，采用简约透明 UI 设计。

---

## ✨ 特性

- 🎀 **简约透明界面** - 白色半透明卡片，适配各种主题背景
- 🔴 **实时直播状态** - 显示"直播中 - 直播间标题"，30 秒自动刷新
- 📺 **追番列表** - 展示最近追番，竖屏封面，悬停上移效果
- 👤 **完整个人资料** - 头像、等级、粉丝、关注、投稿数
- 🔄 **自动更新** - 定时任务每 5 分钟刷新直播状态
- 🛡️ **隐私兼容** - 数据为空时提示检查 B 站隐私设置

## 📦 安装

### 1. 创建插件文件夹

在 `wp-content/plugins/` 下创建：

```
kina-bilibili-integration/
├── kina-bilibili-integration.php
└── assets/
    ├── css/
    │   ├── bili-style.css
    │   └── admin.css
    └── js/
        ├── bili.js
        └── admin.js
```

### 2. 复制代码

将对应代码复制到每个文件中（从 Release 下载 zip 解压即可）。

### 3. 激活插件

进入 WordPress 后台 → 插件 → 安装插件 → 激活 **"Kina Bilibili Integration"**。

### 4. 配置 B 站 UID

进入插件设置页面（左侧菜单 **B 站集成**）：

- 填写你的 B 站 UID（纯数字，从空间地址 `space.bilibili.com/xxxxxx` 获取）
- 点击**保存设置**
- 点击**测试连接**确认正常

### 5. 设置 B 站隐私（重要）

进入 [B 站隐私设置](https://space.bilibili.com) → 设置 → 隐私设置：

- 公开我的关注列表
- 公开我的粉丝列表
- 公开我的追番列表

否则部分数据无法显示。

## 📝 短代码

### `[kina_bilibili_profile]` - 完整个人资料

显示头像、等级、直播状态、粉丝/关注/投稿数、个性签名。

```
[kina_bilibili_profile]
```

**效果：**

- 白色半透明卡片
- 头像带粉色边框
- 直播中红色 LIVE 徽章（弹跳动画）
- 等级星级显示
- 统计数字（关注 / 粉丝 / 投稿）
- 个性签名
- 访问 B 站主页按钮

### `[kina_bilibili_bangumi]` - 追番列表

展示最近追番，竖屏封面网格。

```
[kina_bilibili_bangumi]
```

**效果：**

- 最多 12 部追番
- 3:4 竖屏封面
- 地区、集数标签
- 观看进度角标
- 悬停上移 + 阴影效果

### `[kina_bilibili_status]` - 迷你状态

紧凑状态显示，适合侧边栏或导航。

```
[kina_bilibili_status]
```

**效果：**

- 小头像 + LIVE 徽章
- 昵称 + 粉丝数
- 直播中红色脉冲状态
- 实时更新（30 秒轮询）

## 🔧 常见问题

**Q: 数据为空？**

A: 检查 B 站隐私设置，确保相关列表已公开。如果设置了隐私保护，API 会返回空数据。

**Q: 图片加载失败（403）？**

A: 已修复。插件已添加 `referrerpolicy="no-referrer"`，避免 B 站图片防盗链拦截。

**Q: 直播中状态不更新？**

A: 插件每 30 秒自动刷新，也可以手动刷新页面。确保正在直播且直播间信息公开。

**Q: 显示 PHP 警告？**

A: 所有数组访问均已添加 `isset()` 检查。如果还有问题，在 `wp-config.php` 中设置：

```php
define('WP_DEBUG', false);
```
